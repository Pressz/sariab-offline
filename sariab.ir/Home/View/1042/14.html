جلسه ۷ - فرق Visual Studio و Visual Studio Code

شرکت Microsoft یکی از قدرتمند‌ترین ابزار‌های برنامه‌نویسی دنیا رو تولید کرده. سال‌ها این ابزار کاربردی و جامع که فقط روی سیستم‌عامل ویندوز قابل استفاده بود، باعث رشد صنعت نرم‌افزار شد. و حالا مایکروسافت این محصول خوشنامش رو با معماری و برند دیگری اما با فلسفه‌ای مشابه، به اسم Visual Studio Code وارد بازار کرده تا برند Visual Studio رو به چیزی فراتر از یک محیط توسعه مجتمع تبدیل کنه.
